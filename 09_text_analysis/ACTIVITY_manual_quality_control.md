## Quality Control for AI Text

🕒 *Estimated Time: 10-15 minutes*

---

## ✅ Your Task

Learn to manually perform quality control on AI-generated text by counting concepts, detecting patterns, and creating quality control metrics using `stringr` and `dplyr`.

### 🧱 Stage 1: Load and Analyze Sample Text

- [x] Open [`01_manual_quality_control.R`](01_manual_quality_control.R) in your IDE
- [x] Review the script structure and comments to understand the quality control approach
- [x] Run the script to load the sample report text from [`data/sample_reports.txt`](data/sample_reports.txt)
- [x] Observe the concept counts, quality control checks, and text metrics that are calculated

### 🧱 Stage 2: Create Quality Control Metrics Table

- [x] Examine the quality control results table that combines all metrics
- [x] Review how `stringr` functions are used to:
  - Count concept occurrences (`str_count()`)
  - Detect patterns (`str_detect()`)
  - Extract numbers and percentages
- [x] Review how `dplyr` is used to create summary tables
- [x] Run the script and view the comprehensive quality control results
